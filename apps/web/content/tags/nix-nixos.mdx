[**Nix**](https://nixos.org/guides/how-nix-works/) is a purely functional package manager. [**NixOS**](https://nixos.org/guides/how-nix-works/#how-nixos-works) takes Nix a step further, and bases an entire operating system around it, resulting in a purely functional package management system.
